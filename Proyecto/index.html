Pong for 4

Por si alguna vez se te ocurrio que podría pasar si hicieran el clasico juego de pong pero para 4 jugadores en vez de 2, pues este juego trae la respuesta. Hecho en Python (con Pygame como principal herramienta), este juego trae la gran experiencia pong, pero con un toque caótico.

Características

  • Permite 4 jugadores
  • Controles simples
  • No requiere de joysticks ni mandos externos
  • Contador de victorias
  • Dificultad progresiva a la duración de la ronda
  • Victoria por eliminación
  • Es gratis

Requisitos

  • Tener Python interpreter descargado
  • Tener Pygame descargado
  • Tener una ram (probablemente)
