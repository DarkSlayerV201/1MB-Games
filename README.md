# 1MB Games

## Overview
1MB Games is a personal web project created as my first Front-End experience.  
It started as a project for my Front-End course at *Talento Tech* and later evolved into a space to experiment with web technologies.

The website is intended to showcase small videogame projects and, in the future, allow users to submit their own games through an interactive form.

---

## Features
- Landing page presenting the project concept
- Detailed section for the game **Pong For 4**
- Interactive form with client-side validations
- Email submission via external service
- Deployed and publicly accessible website

---

## Technologies Used
- **HTML**
- **CSS**
- **JavaScript** (form logic and validations)
- **Netlify** (deployment)
- **Formspree** (form handling)

---

## Project Status
The project is functionally complete but still unpolished in terms of UI/UX.  
It is not currently under active development, as I am focusing on improving my Back-End skills.
